# Prime Numbers Finder (1 to N) 🔢

A C++ program that reads a positive number from the user and prints all prime numbers from 1 up to that number. This project focuses on **Algorithms** and **Functional Programming** concepts.

## 🚀 Features
* Validates user input to ensure it's a positive number.
* Uses an optimized logic to check for prime numbers.
* Displays a clear list of all prime numbers in the specified range.

## 🛠️ Concepts Covered
* Loops and Control Flow.
* Functions and Enums.
* Mathematical logic in programming.

## 📖 How to Run
1. Clone the repository.
2. Compile the `main.cpp` file using any C++ compiler (like GCC or Clang).
3. Run the executable and enter a positive number.
