## 📌 Overview

This document explains the concept of a **Database** and the role of **DBMS (Database Management Systems)**.

## 🧠 What is a Database?

A **Database** is an organized collection of data that can be easily accessed, managed, and updated.

To manage databases, we use **DBMS (Database Management Systems)**.

---

## ⚙️ What is DBMS?

A **DBMS** is software that allows users to:

* Store data
* Retrieve data
* Manage data efficiently

---

## ⚔️ Types of Databases

### 🔹 Non-Relational (NoSQL)

* File System
* XML
* Flexible structure

### 🔹 Relational (RDBMS)

* Enhanced version of DBMS
* Supports relationships between data

### 💻 Examples:

* MySQL
* Oracle
* SQL Server

---

## 🔑 Key Concepts

* Database = Organized data
* DBMS = Tool to manage data
* RDBMS = Database with relationships

---

## 📝 Notes

* Databases help in storing large amounts of data
* Data can be accessed easily
* DBMS controls operations on data

---

## 🚀 Conclusion

Databases are a core component in software systems, enabling efficient data storage and management.
