# C++ Debugging Training - Step Into / Over / Out

## 📌 Overview
This project provides hands-on practice for understanding:

- Step Into (F11)
- Step Over (F10)
- Step Out (Shift + F11)

---

## 🧪 Training Idea
Run the program in Debug Mode and control execution using the three debugging tools.

---

## 🎯 Key Concept

- Step Into → go inside function  
- Step Over → execute without entering  
- Step Out → exit function  

---

## 🧠 Quiz (7 Levels)

Try to answer before checking solutions.

1. Enter function → Step Into  
2. Skip known function → Step Over  
3. Debug inside function → Step Into  
4. Exit function → Step Out  
5. Skip function → Step Over  
6. Exit quickly → Step Out  
7. Line by line → Step Into  

---

## 🚀 Goal
Understand when to use each debugging step instead of memorizing.
