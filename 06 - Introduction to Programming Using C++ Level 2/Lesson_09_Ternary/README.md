# C++ Ternary Operator

## 📌 Overview
The ternary operator is a short way to write if/else in one line.

---

## 🔧 Syntax

```cpp
condition ? true : false;
```

---

## 🧠 Example

```cpp
int x = 10;
string result = (x > 0) ? "Positive" : "Negative";
```

---

## ⚠️ When to Use

- ✔️ Simple conditions  
- ❌ Complex logic  

---

## 🔥 Key Idea

Ternary operator returns a value (expression), not just executes code.
