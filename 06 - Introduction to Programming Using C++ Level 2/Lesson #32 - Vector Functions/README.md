# 📘 C++ Lesson 32 — Vector Functions

---

## 📌 Overview

This lesson covers important vector functions.

---

## 🔹 Functions

### First Element

```cpp
v.front();
```

### Last Element

```cpp
v.back();
```

---

### Size

```cpp
v.size();
```

✔️ number of elements

---

### Capacity

```cpp
v.capacity();
```

✔️ reserved memory

---

### Empty

```cpp
v.empty();
```

✔️ returns true/false

---

## 🧠 الفرق المهم

```text
size ≠ capacity
```
