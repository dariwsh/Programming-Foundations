## 📌 Description

A C++ program that:

* Stores employee data using `vector` + `struct`
* Finds the employee with the highest salary
* Displays the result

---

## 🧠 Concepts Used

* `vector<struct>`
* `push_back()`
* Looping through objects
* Comparison logic

---

## 🔹 Data Structure

```cpp
struct stEmployee {
    string FirstName;
    string LastName;
    float Salary;
};
```

---

## 🔹 Features

* Add employees
* Store data dynamically
* Find max salary
* Display top employee

---

```

5000
```
