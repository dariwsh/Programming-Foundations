# C++ Range-Based Loop

## 📌 Overview
Range-based loop allows iterating over elements without using index.

---

## 🔧 Syntax

```cpp
for(type var : array)
```

---

## 🧠 Example

```cpp
int arr[] = {1,2,3};

for(int x : arr)
{
    cout << x;
}
```

---

## ⚠️ Important

- var is a copy
- use & to modify original values

---

## 🎯 Key Idea

Clean and simple loop for iterating elements.
